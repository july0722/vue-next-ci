# web（vue3）

## Project setup
```
yarn install
cd web
```

### Compiles and hot-reloads for development
```
yarn serve demo // or demo-b
```

### Compiles and minifies for production
```
yarn build demo // or demo-b
```

### Lints and fixes files
```
yarn lint
```
